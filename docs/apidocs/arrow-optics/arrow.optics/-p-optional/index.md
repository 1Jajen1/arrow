# POptional

`@higherkind interface POptional<S, T, A, B> : `[`POptionalOf`](../-p-optional-of.html)`<`[`S`](index.html#S)`, `[`T`](index.html#T)`, `[`A`](index.html#A)`, `[`B`](index.html#B)`>`

An [Optional](../-optional.html) is an optic that allows to see into a structure and getting, setting or modifying an optional focus.

A (polymorphic) [POptional](./index.html) is useful when setting or modifying a value for a type with a optional polymorphic focus
i.e. POptional&lt;Ior&lt;Int, Double&gt;, Ior&lt;String, Double&gt;, Int, String&gt;

A [POptional](./index.html) can be seen as a weaker [Lens](../-lens.html) and [Prism](../-prism.html) and combines their weakest functions:

* `set: (S, B) -> T` meaning we can focus into an `S` and set a value `B` for a target `A` and obtain a modified source `T`
* `getOrModify: (S) -> Either<T, A>` meaning it returns the focus of a [POptional](./index.html) OR the original value

### Parameters

`S` - the source of a [POptional](./index.html)

`T` - the modified source of a [POptional](./index.html)

`A` - the focus of a [POptional](./index.html)

`B` - the modified focus of a [POptional](./index.html)
