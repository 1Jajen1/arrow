# PLens

`@higherkind interface PLens<S, T, A, B> : `[`PLensOf`](../-p-lens-of.html)`<`[`S`](index.html#S)`, `[`T`](index.html#T)`, `[`A`](index.html#A)`, `[`B`](index.html#B)`>`

A [Lens](../-lens.html) (or Functional Reference) is an optic that can focus into a structure for
getting, setting or modifying the focus (target).

A (polymorphic) [PLens](./index.html) is useful when setting or modifying a value for a constructed type
i.e. PLens&lt;Tuple2&lt;Double, Int&gt;, Tuple2&lt;String, Int&gt;, Double, String&gt;

A [PLens](./index.html) can be seen as a pair of functions:

* `get: (S) -> A` meaning we can focus into an `S` and extract an `A`
* `set: (B) -> (S) -> T` meaning we can focus into an `S` and set a value `B` for a target `A` and obtain a modified source `T`

### Parameters

`S` - the source of a [PLens](./index.html)

`T` - the modified source of a [PLens](./index.html)

`A` - the focus of a [PLens](./index.html)

`B` - the modified focus of a [PLens](./index.html)
